# Bengkel HP Erman

Website company profile dan katalog produk untuk **Bengkel HP Erman**, layanan service smartphone profesional dan tempat jual beli HP second berkualitas.

## Fitur Utama
- **Desain Premium & Modern**: Tema gelap yang elegan dengan efek *glassmorphism*.
- **Katalog Produk HP Second**: Filter produk berdasarkan merk (Samsung, Oppo, Xiaomi, Vivo, iPhone).
- **Layanan Service Terlengkap**: Daftar layanan service dengan estimasi harga dan waktu pengerjaan.
- **Testimoni Pelanggan**: Integrasi review dari pelanggan yang puas.
- **Integrasi WhatsApp**: Tombol call-to-action yang langsung mengarah ke chat WhatsApp bengkel.
- **Fully Responsive**: Tampilan yang optimal di perangkat desktop, tablet, maupun mobile.

## Teknologi yang Digunakan
Website ini dibangun dari awal menggunakan teknologi web standar tanpa framework eksternal untuk menjaga kecepatan dan efisiensi:
- **HTML5** untuk struktur halaman
- **CSS3** (Vanilla CSS) untuk styling, *glassmorphism*, dan animasi
- **JavaScript** (Vanilla JS) untuk interaktivitas (filter produk, navigasi, *scroll reveal*)
- **Font Awesome 6** untuk ikon-ikon antarmuka
- **Google Fonts** (Plus Jakarta Sans & Syne) untuk tipografi premium

## Cara Menjalankan secara Lokal
1. Clone repositori ini:
   ```bash
   git clone https://github.com/Erman4u/Bengkel-Hp-Erman.git
   ```
2. Buka folder proyek.
3. Cukup buka file `index.html` di browser web favorit Anda (tidak diperlukan server lokal khusus karena hanya menggunakan HTML/CSS/JS statis, namun direkomendasikan menggunakan ekstensi "Live Server" pada VS Code untuk pengalaman pengembangan terbaik).

## Struktur File
- `index.html`: Struktur utama website
- `style.css`: Semua aturan gaya visual dan animasi
- `script.js`: Logika interaktif website
- `hero_technician.png`: Gambar hero section utama
- `about_shop.png`: Gambar ilustrasi deskripsi bengkel
- `products_bg.png`: Placeholder premium untuk gambar produk
